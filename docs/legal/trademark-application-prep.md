### Filing Strategy

1. **New Zealand (IPONZ) — File first**
   - Portal: https://www.iponz.govt.nz/
   - Fee: ~$100 NZD per class
   - This establishes the priority date
   - NZ registration typically takes 6-12 months if no opposition

2. **United States (USPTO) — File directly**
   - Portal: https://www.uspto.gov/trademarks/apply
   - Fee: $250 USD per class (TEAS Plus) or $350 USD per class (TEAS Standard)
   - Use intent-to-use (Section 1(b)) if not yet in commerce in the US
   - Use actual use (Section 1(a)) if already selling to US customers
   - US registration typically takes 12-18 months

3. **International (Madrid Protocol) — File via IPONZ**
   - Portal: https://www.wipo.int/madrid/en/
   - Base fee: 653 CHF
   - Designate: Australia, United Kingdom, European Union
   - Per-country supplementary fees apply
   - Must be based on NZ application or registration

## MARK 2: "THE ORACLE" (CONDITIONAL — PENDING CONFLICT SEARCH)

### ⚠ CONFLICT RISK

"The Oracle" is a common term with existing registrations, most notably
Oracle Corporation's extensive trademark portfolio. A thorough conflict
search must be completed before filing.

### Required Searches Before Filing

| Search | Portal | What to look for |
|---|---|---|
| USPTO TESS | https://tess2.uspto.gov/ | "Oracle" in Classes 42, 45, 36 |
| IPONZ | https://www.iponz.govt.nz/ | "Oracle" in relevant classes |
| WIPO Global Brand Database | https://branddb.wipo.int/ | "Oracle" internationally |
| EUIPO TMview | https://www.tmdn.org/tmview/ | "Oracle" in EU |
| IP Australia (ATMOSS) | https://search.ipaustralia.gov.au/trademarks/search/ | "Oracle" in AU |

### Risk Assessment

- **High risk:** "The Oracle" alone in Class 42 (software) will almost
  certainly conflict with Oracle Corporation's marks
- **Medium risk:** "The Oracle" with a distinctive qualifier may be
  distinguishable
- **Lower risk:** "AlecRae Oracle" or "Oracle by AlecRae" narrows the
  scope and associates with the parent brand

### Alternative Marks (If Conflict Found)

| Alternative | Pros | Cons |
|---|---|---|
| The AlecRae Oracle | Strong parent brand association, more distinctive | Longer, less punchy |
| Oracle by AlecRae | Clear attribution, differentiates from Oracle Corp | Still contains "Oracle" |
| The Oracle™ (with always using alongside AlecRae) | Maintains current branding | Weaker standalone protection |
